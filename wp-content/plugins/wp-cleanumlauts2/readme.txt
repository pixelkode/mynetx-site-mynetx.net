== Description ==

Converts German umlauts for permalinks, post, comments and feeds automatically into their corresponding equivalent.<br>
Wandelt Umlaute automatisch lesbar für Permalinks um.<br>
ä -> ae<br>
ö -> oe<br>
ü -> ue<br>
Ä -> Ae<br>
Ö -> Oe<br>
Ü -> Ue<br>
ß -> ss<br>
<br>
Parts based on wp-cleanumlauts from www.4null4.de<br>
<br>
== Installation ==

1. Upload `wp-cleanumlauts2.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make your settings
done

== Remove plugin ==

1. Deactivate plugin through the 'Plugins' menu in WordPress
2. Delete plugin through the 'Plugins' menu in WordPress
